# ShieldLegal

Landing page for ShieldLegal - Criminal defense services for foreigners in Colombia.

## Project structure

- Static HTML project (single-page landing)
- All CSS and JS are inline in `index.html`
- No build tools, bundlers, or package manager required
- Hosted at https://shieldlegal.co

## Conventions

- Language: English (content targets English-speaking foreigners)
- Fonts: Cormorant Garamond (headings), DM Sans (body)
- Color palette: dark theme with gold accents (see CSS `:root` tokens)
- Mobile-first responsive design

## Git

- Main branch: `main`
- Commit messages: conventional commits (feat:, fix:, docs:, etc.)
